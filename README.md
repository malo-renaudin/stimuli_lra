# stimuli_lra

## Python virtual environment

Create and activate the project's virtual environment:

- Create venv:

```powershell
python -m venv .venv
```

- Activate (PowerShell):

```powershell
.\.venv\Scripts\Activate.ps1
```

- Activate (cmd.exe):

```cmd
.\.venv\Scripts\activate.bat
```

- Install dependencies (if you have a requirements file):

```powershell
.\.venv\Scripts\python -m pip install -r requirements.txt
```

The virtual environment directory `.venv/` is ignored by git.
